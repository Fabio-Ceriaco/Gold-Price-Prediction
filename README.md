# Gold-Price-Prediction

This project aims to predict the price of gold (GLD) based on various financial indicators
such as S&P 500 index (SPX), Crude Oil price (USO), Silver price (SLV), and Euro to USD exchange rate (EUR/USD).
I will use a Random Forest Regressor model for this purpose.


# Stack 

  - Python

# Libraries

  - Numpy
  - Pandas
  - Matplotlib
  - Seaborn
  - Sklearn


# Machien Learning Model

  - Random Forest Regressor
